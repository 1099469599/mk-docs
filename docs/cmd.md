# mk常用命令

## 安装

```
$ npm i -g mk-tools
```

## 创建空网站

```
$ mk website [website name，可空]
```

## 创建应用

```
$ mk app [app name, 可空]
```

## 克隆应用

```
$ mk clone [app name, npmjs发布的应用， 可空]
```

## 编译
- 编译会遍历所有目录下app， 生成index.js, mock.js,  apps.less
```
$ mk compile
```
